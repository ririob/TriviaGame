Trivia Game

This is a trivia game. When game is started, player is given a question with multiple answers to choose from. If player gets it right, an image comes on the screen and if player get the answer wrong, the game automatically display the right answer. The game also keeps track of which answer player gets right or wrong. If player fail to answer a question with the given time, game automatically answers the question. At the end of game, Player will be shown how they fared. Good Luck!
